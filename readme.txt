== Description ==

Hides the login pages of your WordPress site. This plugin is a standalone version of the hide backend feature in <a href="https://wordpress.org/plugins/better-wp-security/" target="_blank">iThemes Security</a> and <a href="https://ithemes.com/security/" target="_blank">iThemes Security Pro.</a>

= Features =

* Hides the login pages of your WordPress site making them only accessible through the address you provide

= Translations =

* English

== Installation ==

1. Backup your Wordpress database, config file, and .htaccess file
2. Upload the zip file to the `/wp-content/plugins/` directory
3. Unzip
4. Activate the plugin through the 'Plugins' menu in WordPress
